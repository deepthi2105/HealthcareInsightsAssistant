# 🏥 Healthcare Insights Assistant

An AI-powered assistant that helps analyze patient health data, visualize lab trends, detect risk factors, and recommend personalized lifestyle improvements — all through natural language queries.

👉 **Try it Live:** [Streamlit App](https://your-streamlit-app-link-here)

---

## 🚀 Features

- 🔍 **Ask in Plain English**  
  "Summarize patient 1", "Visualize cholesterol trends", "Suggest lifestyle changes for Jane Smith"

- 📊 **Lab Trend Analysis**  
  Generates time-series charts and summaries for metrics like HbA1c, Cholesterol, Glucose, etc.

- ⚠️ **Risk Factor Detection**  
  Detects red flags based on test thresholds (e.g., HbA1c ≥ 7, Cholesterol > 200)

- 💊 **Medication & Visit History**  
  Retrieves treatment plans and doctor visit summaries

- 🧠 **Lifestyle Recommendations**  
  Tailored suggestions based on diagnoses and lab patterns
